# Journey

A short Javascript and CSS presentation ([Live Demo](http://nihey.github.io/experiments/journey/))

# Building

This project relies on [webpack][webpack_link] bundling

Production Build:

```
$ npm i
$ npm run production-build
```

Development Build

```
$ npm i -g webpack
$ npm i
$ webpack --watch
```
